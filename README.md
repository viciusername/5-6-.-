## Лабораторна робота 5: Молекулярна динаміка

### Опис
Симуляція взаємодії частинок в 2D просторі з використанням потенціалу Леннарда-Джонса. Моделюються:
- Рух частинок під впливом міжмолекулярних сил
- Збереження енергії в системі
- Періодичні граничні умови

### Запуск

```bash
cd lab5_molecular_dynamics
python molecular_dynamics.py
```

### Результати
Програма створює:
- `energy_plot.png` - графік збереження енергії
- `particles_snapshot.png` - розташування частинок

### Фізичні принципи
- Потенціал Леннарда-Джонса для міжмолекулярної взаємодії
- Алгоритм Velocity Verlet для інтегрування рівнянь руху
- Періодичні граничні умови для симуляції нескінченної системи

## Теоретичні основи

### Лабораторна 5
Молекулярна динаміка моделює рух атомів та молекул шляхом розв'язання класичних рівнянь руху Ньютона. Потенціал Леннарда-Джонса описує взаємодію між нейтральними атомами:

V(r) = 4ε[(σ/r)¹² - (σ/r)⁶]

де ε - глибина потенціальної ями, σ - відстань, на якій потенціал дорівнює нулю.

### Лабораторна 6
Метод Монте-Карло використовує випадкове семплування для розв'язання математичних та фізичних задач. Основні застосування:
- Обчислення інтегралів
- Моделювання випадкових процесів
- Статистична механіка
- Оптимізація
