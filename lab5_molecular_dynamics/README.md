## Мета роботи
Вивчити основи молекулярної динаміки та створити симуляцію взаємодії частинок з використанням потенціалу Леннарда-Джонса.

## Теоретичні відомості

### Потенціал Леннарда-Джонса
Потенціал Леннарда-Джонса описує міжмолекулярну взаємодію:

```
V(r) = 4ε[(σ/r)¹² - (σ/r)⁶]
```

де:
- `r` - відстань між частинками
- `ε` - глибина потенціальної ями (енергія зв'язку)
- `σ` - відстань, на якій потенціал дорівнює нулю

Сила, що діє на частинку:
```
F(r) = -dV/dr
```

### Алгоритм Velocity Verlet
Для інтегрування рівнянь руху використовується алгоритм Velocity Verlet:

1. Обчислити положення в наступний момент часу:
   ```
   r(t + Δt) = r(t) + v(t)Δt + (1/2)a(t)Δt²
   ```

2. Обчислити прискорення в новому положенні:
   ```
   a(t + Δt) = F(t + Δt)/m
   ```

3. Оновити швидкості:
   ```
   v(t + Δt) = v(t) + (1/2)[a(t) + a(t + Δt)]Δt
   ```

### Періодичні граничні умови
Для симуляції нескінченної системи використовуються періодичні граничні умови:
- Частинка, що виходить з одного боку коробки, з'являється з протилежного боку
- Відстань між частинками обчислюється з урахуванням періодичності
