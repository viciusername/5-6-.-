# ЛР3 + ЛР4 — Імітаційне моделювання складальних дільниць (Python)

Один проєкт для двох лабораторних:

- **ЛР3:** одна складальна дільниця: попередня обробка (для 50% деталей) → складання → регулювання (логонормальний час) + події браку/заміни.
- **ЛР4:** цех із **4 дільниць** (4 канали), **одна спільна FIFO-черга** для партій. Пошук такого середнього інтервалу надходження **X** (хв), щоб **максимальна черга за 8 год** не перевищувала 20.

## Вимоги
- Python 3.10+

## Запуск (Windows PowerShell)
```powershell
cd lab34_factory_sim
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m src.run_all
```

## Що генерується
Після запуску всі результати з'являться у папці `outputs/`:
- `lab3_summary.csv`, `lab3_plots_*.png`
- `lab4_search_x_results.csv`, `lab4_plots_*.png`
- `lab4_analytic_vs_mc.csv`

За потреби можете змінити параметри у `src/config.py`.
